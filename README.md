패키지
---
clean.js의 패키지 구조는 다음과 같애요.
- array : 배열을 처리하는 기능들이 있어요.
- object : 객체를 처리하는 기능들이 있어요.
- date : 날짜를 처리하는...
- string : 문자열을... ㅋㅋ

테스트 방법
---
1. source에 코드를 작성해요!
2. root 폴더에서 node build를 입력해요! 그럼 clean.js가 새로 만들어집니다!
3. test 폴더에 테스트용 html파일을 아무렇게나 작성하여 실행합니다!!
